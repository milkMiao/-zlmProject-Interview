CSS面试高频题合集-----BFC

1、哪4种情况会形成BFC渲染区域

• float的值不是none
• position的值不是static或者relative。
• display的值是inline-block、table-cell、flex、table-caption或者 inline-flex
• overflow的值不是visible

2、解决垂直方向margin合并

总结: 解决垂直方向margin合并
• Step1: 添加父元素包裹下方元素
• Step2:
• 父元素overflow:hidden
• 父元素下第一个子元素前添加空table标签
• 父元素padding代替子元素margin
• 父元素+透明上边框
• 父元素::before{ content:””; display:table }

3、解决垂直方向margin溢出

总结: 解决垂直方向margin溢出
• 父元素overflow:hidden
• 父元素下第一个子元素前添加空的table标签
• 父元素padding代替子元素margin
• 父元素+透明上边框
• 父元素::before{ content:””; display:table }

4、左边固定宽度，右边自适应

总结: 左定宽，右自适应布局
• Step1: 左: float:left
• Step2: 右: 右overflow:hidden
